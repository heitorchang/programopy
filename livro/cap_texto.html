3. Texto

Você quer manter mais controle sobre sua biblioteca, então decide criar variáveis para se referirem ao nome do autor.

Um valor textual se chama uma string. Valores string começam e terminam com aspas.

Não há diferença escolher entre aspas simples 'programar' ou aspas duplas "programar". Dentro da string, se tiver que incluir aspas do mesmo tipo, terá que digitar \ antes das aspas: "A ilha se chamava \"Formosa\" antes.".
