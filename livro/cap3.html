3. Escolha seu ambiente

Até agora, mostrei como programar em Python nessa página mesmo. Porém, é um ambiente bastante limitado, que não tem capacidade de armazenar o seu trabalho, por exemplo.

Existem programas chamados IDEs (sigla em inglês que significa "Ambiente de desenvolvimento integrado") que ajudam na organização e verificação do seu código.

O interpretador e blocos de código continuarão a existir nos capítulos seguintes, caso você não queira usar um IDE nesse momento.

Entre todos os IDEs que conheço, recomendo três:

  • PyCharm Community Edition
  • Microsoft Visual Studio Code
  • Thonny

Os dois primeiros são usados por profissionais, e o último foi projetado numa universidade sobretudo para principiantes.

Além de instalar o IDE, você terá que instalar Python no seu computador se ele já não estiver presente.

Siga instruções online caso algo não funcionar. Infelizmente, não sou capaz de oferecer soluções para todo tipo de configuração.
