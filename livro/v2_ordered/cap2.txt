$h1 2. Variáveis

Além de ler o bom livro de Python, você planeja ler outros livros, sobre <i>JavaScript</i>, <i>PHP</i>, e <i>C</i>.

Cada livro tem a quantidade de páginas dada abaixo, e você quer saber quantas páginas você vai ler. Clique no botão amarelo <b>Avalie</b> para executar o código.

@
javascript = 520
php = 428
c = 922

javascript + php + c
/

As primeiras três linhas criam variáveis. Uma <b>variável</b> é um nome que se refere a um valor. Cada nome de linguagem se refere a quantas páginas aquele livro contém.

Quando avaliada, uma variável devolve o valor a qual ela se refere. A <b>expressão</b> `javascript + php + c` é uma combinação de variáveis e o operador `+`, que faz somas. Essa expressão é igual a `520 + 428 + 922`, que é `1870`.

Agora, você quer adicionar o número de páginas do livro de Python. Ele tem `870`páginas. Clique na caixa preta acima e adicione: `python = 870`. Altere também a soma, para `javascript + php + c + python`.

Finalmente, digite `Shift + Enter` (segure `Shift` e tecle `Enter`). Ou, poderá também clicar no botão <b>Avalie</b>.

Você terá lido `2740` páginas!
