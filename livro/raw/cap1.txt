$h3 Calculadora para uma festinha

@py
def calcule_total(núm_pizzas, núm_refris):
    valor_pizza = 50
    valor_refri = 15

    return (núm_pizzas * valor_pizza) + (núm_refris * valor_refri)
@endpy

$p Repare que no nome das variáveis, acentuei <code>núm</code>. Python também permite que use caracteres especiais como <code>π</code> em nomes, que podem facilitar a leitura. Lembre-se que cada vez que usar o nome, terá que digitá-lo exatamente da mesma forma.

@py
aprox_π = 3.14
print(2 * aprox_π)
@endpy
