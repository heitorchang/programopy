4. Escolha seu ambiente

Até agora, mostrei como escrever Python neste site. Porém, é um ambiente bastante limitado, que não consegue gravar seu trabalho, por exemplo.

Existem programas chamados IDEs (Ambiente de desenvolvimento integrado) que organizam e verificam seu código.

Entre os IDEs que conheço, recomendo três:

  • PyCharm Community Edition
  • Microsoft Visual Studio Code
  • Thonny

Os dois primeiros são usados por profissionais, e o último foi projetado numa universidade, principalmente para ser usado por principiantes.

Siga instruções online caso algo não funcionar. Infelizmente, não sou capaz de oferecer soluções para todo tipo de configuração.

O interpretador e blocos de código deste site continuarão a existir nos capítulos seguintes, caso você não queira usar um IDE nesse momento.
