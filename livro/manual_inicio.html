O que é Python?

Python é uma linguagem de programação famosa pela sua clareza, versatilidade, e extensa coleção de pacotes contribuídos por grandes empresas e indivíduos, servindo para milhares de utilidades.

Ela é uma linguagem interpretada, que significa que um trecho de código pode ser executado imediatamente, sem obrigar o programador a pausar para compilar seu código (transformar os símbolos do código em linguagem de máquina), como é feito nas linguagens compiladas.

Sobre este livro

Este livro foi projetado para quem não tem muita experiência de programação e deseja desenvolver a competência para criar programas úteis e entender o código de outros programadores.

Explico as principais características e funcionalidades de Python de maneira acessível, usando diversos exemplos e evitando complicações desnecessárias.

Para experimentar a escrever código Python imediatamente, montei uma interface web com um interpretador Python e um editor para manipular trechos curtos de código.

Requisitos

Para interagir com o interpretador e editor, é recomendado usar um ambiente de desktop. A interface foi testada com Google Chrome (versão 121).

Não foram feitos teste de usabilidade num ambiente móvel (em smartphones), e não tenho recursos suficientes para garantir a operação correta nos smartphones mais populares.

Como escrever Python nesta página

Na parte inferior da página está o interpretador interativo de Python. Os três sinais de "maior que" >>> indicam que o interpretador está pronto para receber seu código.

Experimente clicar na janela preta abaixo, e digite 1 + 2. Para pedir que o interpretador avalie (isto é, produza um valor para) esta expressão, tecle Enter (também chamado de Return).

Você verá o resultado, 3, numa linha abaixo.

No conteúdo deste livro, haverão blocos editáveis de código, que terão esta aparência:

Clique dentro desse editor para ativar o cursor. Com o cursor dentro do editor, segure Shift e tecle Enter (também chamado de Return) para enviar este código ao interpretador. Esta combinação é frequentemente escrita Shift + Enter.

Se preferir, clicando no botão amarelo Avalie terá o mesmo efeito.

Outros blocos contém código independente, mas uma vez enviada ao interpretador, eles permanecerão lá. (Observe que reutilizei a variável b do primeiro bloco, e se este segundo bloco ser avaliado primeiro, o interpretador dirá que b não foi definido).

Você poderá fazer alterações ao código, e quando terminar, tecle Shift + Enter para que o interpretador receba as atualizações.
