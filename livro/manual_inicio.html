Início

O que é Python?

Python é uma linguagem de programação famosa pela sua clareza, versatilidade, e extensa coleção de pacotes contribuídos por grandes empresas e indivíduos, servindo para milhares de utilidades.

Ela é uma linguagem interpretada, que significa que um trecho de código pode ser executado imediatamente, sem obrigar o programador a pausar para compilar seu código (transformar os símbolos do código em linguagem de máquina), como acontece nas linguagens compiladas.

Por que ler este livro

Já existem dezenas de livros bons sobre Python. Como este livro é diferente? Este livro foi projetado para quem não tem muita experiência de programação, mas deseja desenvolver a competência para criar programas úteis e entender o código de outros programadores.

Meu objetivo é explicar as principais características e funcionalidades de Python de maneira acessível, usando diversos exemplos e evitando complicações desnecessárias.

Após terminar a leitura, desejo que você tenha a confiança para projetar programas úteis, seguir tutoriais de bibliotecas do seu interesse, entender o código de outros, e continuar sua jornada com livros mais avançados.

Para experimentar a escrever código Python imediatamente, esta página inclue um interpretador Python e é possível editar trechos curtos de Python com facilidade.

Requisitos

Esta página deve ser usada num ambiente de desktop, e foi testada com Google Chrome (versão 121).

Num ambiente móvel (smartphone), o interpretador e editor não funcionam corretamente.

Use um computador desktop se gostaria de praticar a escrever código usando esta interface.

Como escrever Python nesta página

Na parte inferior da página está o interpretador interativo de Python. Os três sinais de "maior que" >>> indicam que o interpretador está pronto para receber seu código.

Experimente clicar na janela preta abaixo, e digite 1 + 2. Para pedir que o interpretador avalie (calcular) esta expressão, tecle Enter (também chamado de Return).

Você verá o resultado, 3, numa linha abaixo.

Clique dentro desse editor para ativar o cursor. Com o cursor dentro do editor, segure Shift e tecle Enter (também chamado de Return) para enviar este código ao interpretador. Esta combinação é frequentemente escrita Shift + Enter.

Outros blocos contém código independente, mas uma vez enviada ao interpretador, eles permanecerão lá. (Observe que reutilizei a variável b do primeiro bloco, e se este segundo bloco ser avaliado primeiro, o interpretador dirá que a não foi definido).

Você poderá fazer alterações ao código, e quando terminar, tecle Shift + Enter para que o interpretador receba as atualizações.
