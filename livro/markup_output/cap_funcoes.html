4. Funções

Para explicar o que são todas essas variáveis, você pensa juntar a informação numa string só. O operador +, quando usado em strings, junta eles.

Repare que é um pouco entediante ter que criar variáveis novas para cada livro. E se você quiser mudar a string, teria que mudar todas elas.

Uma função aceita valores, faz operações neles, e devolve qualquer número de valores (de zero a números astronômicos, limitado apenas pela memória do seu computador).

Se os valores como números e texto são substantivos, funções são como verbos que agem sobre os valores.

A definição de uma função começa com a palavra-chave def, seguido pelo nome dela.

A lista de valores que ela deve aceitar vem entre parênteses.

Finalmente, escreva o sinal de "dois pontos" :.

def faça_algo(coisa, valor):

O corpo da função precisa ser endentado. Isso quer dizer que toda linha do corpo deve começar com quatro espaços. Os quatro espaços são um nível de endentação. A cada nível, quatro espaços a mais devem ser adicionados.

Após definir uma função, ela pode ser chamada. Para chamar uma função, escreva o nome, seguido pela lista de valores passados a ela (esses valores são chamados de argumentos).
