Números

Calculadora para uma festinha

Repare que no nome das variáveis, acentuei núm. Python também permite que use caracteres especiais como π em nomes, que podem facilitar a leitura. Lembre-se que cada vez que usar o nome, terá que digitá-lo exatamente da mesma forma.
